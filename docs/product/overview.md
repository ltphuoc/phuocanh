# Product Overview

## Product
Private mobile-first web app for one couple (exactly two users) to capture memories, plan activities, revisit shared moments, and grow a private visual archive together.

## Target Users
- User A and User B in one shared private space
- No multi-couple tenancy in Phase 1

## Core UX Principles
- Fast capture flow on mobile (note + media in under 30 seconds)
- Privacy by default (auth required, RLS enforced, private storage buckets)
- Story-first home with an emotional featured chapter before utilities
- Minimal-friction navigation with a floating mobile dock and a desktop rail + expandable drawer
- Incremental delivery by phase with production-safe defaults
- Warm, memorable editorial identity with light-only premium surfaces and accessible contrast

## Non-goals (Phase 1)
- Public sharing
- Multi-couple workspaces
- AI generation/agent behavior
- Native mobile app
